# Quick Guide - Role Assignment in Add/Edit

## 🚀 Quick Start

### Add Member with Role (3 steps)
1. Click **+ Add Member**
2. Fill in: Name, ID, Phone, **Select Role**
3. Click **Add Member** ✅

### Edit Member's Role (3 steps)
1. Find member → Click **Edit icon**
2. Change **Role** dropdown
3. Click **Update Member** ✅

---

## 📋 Available Roles

- **No Role** - Regular member
- **Chairperson** - Read-only access
- **Vice Chairperson** - Read-only access
- **Secretary** - Can manage events & minutes
- **Vice Secretary** - Read-only access
- **Patron** - Read-only access
- **Administrator** - Full access

---

### Change to Chairperson
```
Find John Doe → Edit

```
Find John Doe → Edit
Role: No Role ← Select this
