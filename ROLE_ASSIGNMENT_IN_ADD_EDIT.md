# Role Assignment During Member Add/Edit

## Overview

You can now assign roles directly when adding or editing members, without needing to use the separate role assignment dialog.

---

## Adding a Member with Role

### Steps

1. **Go to Members page** (Admin Dashboard → Members)
2. **Click "+ Add Member"** button
3. **Fill in member details**:
   - Name
   - ID Number
   - Phone
4. **Select Role (Optional)**:
   - Click the "Role" dropdown
   - Choose from:
     - No Role (default)
     - Chairperson
     - Vice Chairperson
     - Secretary
     - Vice Secretary
     - Patron
     - Administrator
5. **Click "Add Member"**

The member will be created with the selected role immediately.

---

## Editing a Member's Role

### Steps

1. **Go to Members page** (Admin Dashboard → Members)
2. **Find the member** in the table
3. **Click Edit icon** (pencil)
4. **Update member information** if needed:
   - Name
   - ID Number
   - Phone
5. **Change Role**:
   - Click the "Role" dropdown
   - Select new role or "No Role" to remove
6. **Click "Update Member"**

The member's role will be updated immediately.

---

## What's New

### Before
- Add member → Member created with no role
- Assign role → Use separate Shield icon dialog

### Now
- Add member → Can assign role at same time
- Edit member → Can change role at same time
- No need for separate role assignment dialog

---

## Role Options

| Role | Description |
|------|-------------|
| No Role | Regular member (default) |
| Chairperson | Read-only access to defaulters, payments, members |
| Vice Chairperson | Read-only access to events, stats, penalties |
| Secretary | Can manage events and meeting minutes |
| Vice Secretary | Read-only access to records and documentation |
| Patron | Read-only access to governance and metrics |
| Administrator | Full system access |

## Examples

### Example 1: Add Secretary
1. Click "+ Add Member"
2. Enter: Name = "John Doe", ID = "12345678", Phone = "0712345678"
3. Select Role = "Secretary"
4. Click "Add Member"
5. John is created as a Secretary immediately

### Example 2: Change Member to Chairperson
1. Find member in table
2. Click Edit icon
3. Select Role = "Chairperson"
4. Click "Update Member"
5. Member's role is changed to Chairperson

### Example 3: Remove Member's Role
1. Find member in table
2. Click Edit icon
3. Select Role = "No Role"
4. Click "Update Member"
5. Member's role is removed

## Benefits

---

## Still Available

The separate role assignment dialog (Shield icon) is still available if you prefer:
- Find member in table
- Click Shield icon
- Select role
- Click "Assign Role"

Both methods work the same way.

---

## Notes

- Role assignment is optional when adding members
- Members can have only one role at a time
- Changing role replaces the previous role
- "No Role" means the member is a regular member
- Role takes effect immediately after save
