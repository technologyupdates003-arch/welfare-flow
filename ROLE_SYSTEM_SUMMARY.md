## 🛡️ How to Assign Roles

### Location: Admin Dashboard → Members

```
1. Find member in table
2. Click Shield icon (4th button)
3. Select role from dropdown
4. Click "Assign Role"
5. Done! Role appears in table
```

### Available Roles:
- Chairperson
- Vice Chairperson
- Secretary
- Vice Secretary
- Patron
- Administrator

---

## 📊 What Each Role Sees

### Member (No Role)
- Events, Documents, News, Profile, Beneficiaries

### Chairperson
- Defaulters, Payments, Active Members (read-only)

### Vice Chairperson
- Events, Member Stats, Payments (read-only)

### Secretary
- **Manage Events** (create/edit/delete)
- **Meeting Minutes** (create/edit/delete)

### Vice Secretary
- Documents, Records, Activities (read-only)

### Patron
- Overview, Statistics, Performance (read-only)

### Admin
- **Full system access** (everything)
