## 📋 ROLE ASSIGNMENT (Admin Only)

### Steps to Assign a Role
1. Login as Admin (phone + Member2026)
2. Click **Members** in sidebar
3. Find member using search
4. Click **Shield icon** in Actions column
5. Select role from dropdown
6. Click **Assign Role**

### Available Roles to Assign
- Chairperson
- Vice Chairperson
- Secretary
- Vice Secretary
- Patron
- Administrator

## 👥 MEMBER MANAGEMENT (Admin Only)

### Add New Member
1. Go to Members page
2. Click **+ Add Member** button
3. Enter: Name, ID Number, Phone
4. Click **Add Member**
5. Member can now login with phone + Member2026

### Edit Member
1. Find member in Members table
2. Click **Edit icon** (pencil)
3. Update: Name, ID Number, Phone
4. Click **Update Member**

### Manage Beneficiaries
1. Find member in Members table
2. Click **Users icon**
3. Click **Add Beneficiary** tab
4. Enter: Name, Relationship, Phone (optional), ID (optional)
5. Click **Add Beneficiary**
6. To remove: Click trash icon in Beneficiaries List

### Delete Member
1. Find member in Members table
2. Click **Trash icon**
3. Confirm deletion

## 🔐 ROLE-SPECIFIC DASHBOARDS

### Admin Dashboard
- Full access to all features

- Role assignment
- All reports and data

### Secretary Dashboard
- **Event Management**: Create, edit, delete events

- **View-only**: Defaulters, payments, active members

### Vice Chairperson Dashboard
- **View-only**: Events, member stats, penalties

### Vice Secretary Dashboard
- **View-only**: Records and documentation

### Patron Dashboard
- **View-only**: Governance overview, performance metrics
- No edit permissions

### Member Dashboard
- Personal profile
- View events
- View documents
- View news
- Manage beneficiaries
- View notifications

## 📝 MEETING MINUTES (Secretary Only)

### Create Meeting Minutes
1. Login as Secretary
2. Go to **Meeting Minutes** (in sidebar)
3. Click **+ New Minutes**
4. Fill in:
   - Meeting Date
   - Meeting Type
   - Title
   - Attendees
   - Agenda
   - Discussions
   - Decisions
   - Action Items
   - Next Meeting Date
5. Click **Save**

### Edit Meeting Minutes
1. Go to Meeting Minutes
2. Find minutes in list
3. Click **Edit icon**
4. Update information
5. Click **Save**

### Delete Meeting Minutes
1. Go to Meeting Minutes
2. Find minutes in list
3. Click **Delete icon**
4. Confirm deletion

## 🔍 TROUBLESHOOTING

### "I can't login"
- Check phone number format (0712345678 or +254712345678)
- Password is always: `Member2026`
- Make sure member was created in Members page

### "I don't see my role's dashboard"
- Logout and login again
- Check that role was assigned in Members page
- Verify role appears in Role column

### "I can't assign a role"
- Make sure you're logged in as Admin
- Make sure member was created (has user account)
- Try refreshing the page

### "Meeting Minutes not showing"
- Make sure you have Secretary role
- Go to Secretary Dashboard → Meeting Minutes
- Check that minutes were saved
