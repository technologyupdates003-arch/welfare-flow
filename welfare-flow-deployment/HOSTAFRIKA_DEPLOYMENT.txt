# WELFARE FLOW - HOSTAFRIKA DEPLOYMENT

## FTP Upload Instructions

1. Connect to HostAfrica via FTP
   - Host: your-domain.com
   - Username: your-cpanel-username
   - Password: your-cpanel-password
   - Port: 21

2. Upload Files
   - Upload all files from 'public_html' folder to your HostAfrica public_html directory
   - Ensure .htaccess file is uploaded (hidden file)

3. Verify Upload
   - Check that index.html is in public_html
   - Check that assets folder exists
   - Verify .htaccess is present

4. Configure Environment
   - Create .env file in public_html with:
     VITE_SUPABASE_URL=your_supabase_url
     VITE_SUPABASE_ANON_KEY=your_anon_key

5. Test Application
   - Visit your domain
   - Check browser console for errors
   - Test all features
