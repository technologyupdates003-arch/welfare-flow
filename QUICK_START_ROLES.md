# Quick Start: Role Assignment & Login

## 🚀 Quick Summary

**The system is simple:**
1. All users login with: **Phone + Password (Member2026)**
2. System detects their role
3. User sees their role-specific dashboard
4. **No difference in login** - only dashboard differs

---

## 📋 How to Assign a Role (Admin)

### In 5 Steps:
```
1. Admin Dashboard → Members
2. Find member (search by name/phone/ID)
3. Click Shield icon in Actions column
4. Select role from dropdown
5. Click "Assign Role"
```

**Done!** Member now has that role.

## 📊 What Each Role Sees

| Role | Dashboard | Key Features |
|------|-----------|--------------|
| **Member** | Member | Events, Documents, News, Profile |
| **Chairperson** | Chairperson | Defaulters, Payments (read-only) |
| **Vice Chairperson** | Vice Chairperson | Events, Stats (read-only) |
| **Secretary** | Secretary | **Manage Events, Meeting Minutes** |
| **Vice Secretary** | Vice Secretary | Documents, Records (read-only) |
| **Patron** | Patron | Overview, Statistics (read-only) |
| **Admin** | Admin | **Full System Access** |
